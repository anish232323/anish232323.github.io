# Portfolio Website

Built with Next.js, React, and Framer Motion.

## To Update Content

Edit the files in `C:\Users\anish\Downloads\anish-portfolio\src\data\portfolioData.ts` and rebuild:

```bash
cd C:\Users\anish\Downloads\anish-portfolio
npm run build
```

Then copy the `out` folder contents to this `website` folder and push to GitHub.
